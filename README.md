# CyberGuard AI Defense System

CyberGuard is a real-time cybersecurity monitoring and defense simulation platform.  
It demonstrates how an AI-based system can detect, analyze, and respond to common web attacks using a live monitoring dashboard.

The project includes a demo website that can be attacked and a monitoring system that detects those attacks in real time. The dashboard displays threat levels, attack types, system status, and attack statistics through graphs and panels.

---

## Features

- Real-time attack monitoring
- Brute Force attack simulation
- SQL Injection detection
- DoS attack simulation
- AI-based threat response simulation
- Live cybersecurity dashboard
- Attack activity graph
- Attack distribution graph
- Threat feed log viewer
- Attack statistics panel
